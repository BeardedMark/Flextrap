# Flextrap

Библиотека разметки, основанная на Flexbox и золотых пропорциях (последовательность Фибоначчи).

## Описание

Flextrap — это CSS-библиотека, предоставляющая набор утилитарных классов и компонентов для быстрой и удобной верстки веб-интерфейсов. Библиотека использует принципы Flexbox для создания гибких макетов и последовательность Фибоначчи для гармоничных пропорций размеров и отступов.

## Основные возможности

- **Утилитарные классы** для работы с Flexbox, позиционированием, отступами, типографикой
- **Готовая система цветов** с предустановленной палитрой
- **Компоненты** для создания каркасов интерфейса (frames, shadows, UI элементы)
- **Адаптивная система размеров** на основе золотых пропорций
- **SCSS исходники** для кастомизации под ваши нужды

## Установка

```bash
npm install flextrap
```

## Использование

Подключите скомпилированный CSS файл в ваш проект:

```html
<link rel="stylesheet" href="node_modules/flextrap/dist/style.css">
```

Или импортируйте SCSS исходники для кастомизации:

```scss
@import 'node_modules/flextrap/src/style.scss';
```

## Структура проекта

- `src/` — исходные SCSS файлы
  - `_variables.scss` — переменные (цвета, размеры, шрифты)
  - `utilities/` — утилитарные классы
  - `components/` — готовые компоненты
- `dist/` — скомпилированный CSS

## Разработка

```bash
# Сборка проекта
npm run build

# Режим разработки с отслеживанием изменений
npm run dev
```
